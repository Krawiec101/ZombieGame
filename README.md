# Game

Prosty projekt gry w Pythonie z menu glownym opartym o `pygame`.

## Szybki start gry

```bash
python src/main.py
```

## Wymagania

- Python 3.12+
- `pip`

## Uruchomienie przez Docker Compose

Build i uruchomienie testow:

```bash
docker compose build
docker compose run --rm app pytest
```
